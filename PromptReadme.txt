Crie uma ferramenta de gerenciamento de projetos com as seguintes características:

1. Estrutura de Navegação
Crie uma área principal chamada "Projects" para organização e gerenciamento de projetos.

Adicione uma pasta chamada "Dashboard" com indicadores gerais.

Crie uma nova pasta chamada "Import CSV" para importação de projetos e tarefas via arquivos CSV.

2. Funcionalidades – Projects
Permitir o cadastro de novos projetos com campos: nome, descrição, data de início e data de término.

Exibir lista de projetos cadastrados com visualização simples dentro da pasta "Projects".

Criar botão funcional "New Project" dentro da pasta "Projects", permitindo criação rápida de novos projetos.

3. Funcionalidades – Dashboard
Mostrar os seguintes indicadores resumidos:

Total Projects

Tasks This Week

Completed Tasks

Total Tasks

4. Funcionalidade – Importação CSV
Permitir upload de um arquivo .csv para:

Criar automaticamente um novo projeto.

Criar atividades relacionadas a esse projeto com base nas colunas do CSV.

5. Funcionalidade – Lista de Atividades
Criar uma visualização em lista das atividades de cada projeto.

Exibir para cada atividade:

Nome da atividade

Responsável

Status (em andamento, concluído, pendente)

Data de início

Data de término

6. Funcionalidade – Project Activities (Histórico)
Criar uma visão adicional chamada Project Activities com o histórico de atualizações, interações e comentários nas tarefas dos projetos.
